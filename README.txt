 Pacman AI Game

Bu proje, klasik Pacman oyununu temel alan ve hayaletlerin 
A*, BFS, DFS, subAstar, BFSWithHeuristic gibi algoritmalarla
Pacman'i kovaladığı bir versiyonudur.


Kurulum
Gerekli Paketler
Python 3.x yüklü olmalıdır. 
Gerekli kütüphane:
pip install pygame

Oyunu çalıştırma
python pacman.py

Oyun Kontrolleri
Yön Tuşları: Pacman’i hareket ettirir.
ESC: Oyunu kapatır.
